Accepted file format: .pkl (model and vectorizer).
.pkl files are serialized objects generated by Python's pickle module. They can contain various Python objects, such as models, vectors, or any other pickled objects. In the context of this application, .pkl files store machine learning models and vectorizers necessary for sentiment analysis.

Restart the application

Restarting the application forces the loading of the new model and vectorizer. When the application is restarted, it clears any existing cached data and reloads the necessary components from scratch. This ensures that any updates or changes to the model and vectorizer are applied immediately and accurately, providing the most up-to-date sentiment analysis results for your reviews.
